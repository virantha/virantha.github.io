PyPDFOCR 0.5.2 - bug fixes

Version 0.5.2 of PyPDFOCR is now available on PyPI.

After installing Ghostscript and Tesseract-OCR, you can install and run it by doing:

pip install pypdfocr
pypdfocr

On Windows, I still recommend downloading the prebuilt binary on GitHub: pypdfocr.exe

This release fixes an issue where keywords were not being lower-cased before being compared to the lower-cased document text. It also extracts text that had nested styles (like bold+italic) in the PDF properly now.

As always, any and all feedback is welcome, and you can also look at the source at GitHub
